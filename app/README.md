# App Frontend (Expo)

This directory contains the cross-platform frontend for both web and mobile, built with Expo and React Native.

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the Expo development server:
   ```sh
   npm start
   ```

## Running the App

- **Web:**
  - Press `w` in the terminal or select "Run in web browser" in Expo Dev Tools.
- **iOS Simulator:**
  - Press `i` in the terminal (requires Xcode on Mac).
- **Android Emulator:**
  - Press `a` in the terminal (requires Android Studio).
- **Physical Device:**
  - Install Expo Go from the App Store or Google Play.
  - Scan the QR code shown in the terminal or Expo Dev Tools.

## Project Structure

- `src/` — All source code (components, screens, navigation, etc.)
