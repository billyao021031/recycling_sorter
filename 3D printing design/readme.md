# 3D Printing Design

This folder contains the 3D models and printing configurations used in the Recycling Sorter project.

## Design Software

All parts were designed using FreeCAD.

FreeCAD was used to:

- Create parametric CAD models
- Adjust dimensions to fit sensors, motors, and mechanical components
- Export final models as .STL files for slicing

Key advantages:

- Parametric modeling allows easy modification
- Open-source and compatible with multiple platforms
- Suitable for mechanical component design

## 3D Printer

All parts were printed using the Ender 3 V3 SE.

Typical printer settings:

- Nozzle diameter: 0.4 mm  
- Layer height: 0.2 mm  
- Infill: 20–30%  
- Material: PLA  
- Print temperature: 200°C  
- Bed temperature: 60°C  

These settings provide a good balance between print strength and print time.

## Slicing Software

All STL models were sliced using Ultimaker Cura.

Cura was used to:

- Generate G-code for the printer
- Configure printing parameters
- Add supports when required

Typical Cura settings:

- Layer height: 0.2 mm
- Infill: 20–30%
- Print speed: 50 mm/s
- Support: Enabled when necessary
- Build plate adhesion: Skirt or Brim

## Printing Workflow

The typical workflow used in this project:

1. Design the part in FreeCAD
2. Export the model as STL
3. Import the STL into Cura
4. Configure print parameters
5. Generate G-code
6. Print using the Ender 3 V3 SE

## File Types

| File Type | Description |
|-----------|-------------|
| .FCStd | FreeCAD project files |
| .STL | Exported 3D printable models |
| .gcode | Printer-ready files generated by Cura |
